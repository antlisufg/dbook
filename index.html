HealthDB

A database management system for health information systems

View project on GitHub

Bem-vindo!

HealthDB é um sistema de gerenciamento de banco de dados cuja linguagem de definição de dados é ADL e cuja linguagem de manipulação de dados é AQL. ADL e AQL são produtos da fundação openEHR.
